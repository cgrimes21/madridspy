interact proc (atom)

See also:
get_hit proc
get_hit_self proc
get_hit_hand proc
Click proc (atom)
Format:
interact(who, hand=0)
When:
Called when an atom is clicked on.
Args:
who is the player clicking src
hand is what hand the player is using (left or right) 0 or 1.
Default hand value is zero (indicating a left hand)
Returns:
Returns src.get_hit
Default action:
Get the item that resides in who's hand used to interact with source. If nothing can be found, call src.get_hit_hand(who).

If src is in fact the item in who's hand, call src.get_hit_self(who). If finally all else passed, call src.get_hit(who,who.hand)
